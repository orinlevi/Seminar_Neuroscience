## 🧪 נוסחאות חשובות

### משוואת נרנסט
$$E_{ion} = \frac{RT}{zF} \ln\left(\frac{[ion]_{out}}{[ion]_{in}}\right)$$

### משוואת Goldman-Hodgkin-Katz
$$V_m = \frac{RT}{F} \ln\left(\frac{P_{Na}[Na^+]_{out} + P_K[K^+]_{out} + P_{Cl}[Cl^-]_{in}}{P_{Na}[Na^+]_{in} + P_K[K^+]_{in} + P_{Cl}[Cl^-]_{out}}\right)$$
